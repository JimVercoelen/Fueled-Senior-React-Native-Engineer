1. **Fix Supabase mock** (`tests/__mocks__/supabase.ts`):
   The root cause of the failing useCreateItem test is that `mockSupabase.from()` creates a new chainable on each call, so `mockSupabase.from.mock.results[0]?.value` is undefined when mutations call `from()` inside the mutationFn (async timing).

   Fix approach: Make `from()` return a **stable per-call chainable** that properly captures the reference. The key change is that each CRUD method (insert, update, delete, select) on the object returned by `from()` must also return a chainable that resolves when awaited. The mutation hooks chain calls like `.from('items').insert({...}).select().single()` and `.from('items').delete().eq('id', id)`.

   Specifically:
   - Keep `createChainable()` as-is (it works for query chains)
   - Update `from()` so its return value stores itself on `mockSupabase.from` in a way tests can access. Simplest approach: add a `lastFrom` property on mockSupabase that always holds the most recent `from()` return value:
     ```typescript
     from: Object.assign(jest.fn(() => {
       const chainable = createChainable();
       const tableApi = {
         select: jest.fn().mockReturnValue(chainable),
         insert: jest.fn().mockReturnValue(chainable),
         update: jest.fn().mockReturnValue(chainable),
         delete: jest.fn().mockReturnValue(chainable),
       };
       mockSupabase.lastFrom = tableApi;
       return tableApi;
     }), { lastFrom: null as any })
     ```

2. **Create useCacheEntries test** (`tests/hooks/use-cache-entries.test.ts`):
   ```typescript
   import React from 'react';
   import { renderHook } from '@testing-library/react-native';
   import { QueryClient, QueryClientProvider } from '@tanstack/react-query';
   import { useCacheEntries } from '../../src/hooks/use-cache-entries';

   function createWrapper() {
     const queryClient = new QueryClient({
       defaultOptions: { queries: { retry: false, gcTime: Infinity } },
     });
     // Seed a query so the cache is not empty
     queryClient.setQueryData(['test-key'], { items: [1, 2, 3] });
     return { queryClient, Wrapper: ({ children }: { children: React.ReactNode }) =>
       React.createElement(QueryClientProvider, { client: queryClient }, children) };
   }

   describe('useCacheEntries', () => {
     it('returns cache entries on mount', () => {
       const { Wrapper } = createWrapper();
       const { result } = renderHook(() => useCacheEntries(), { wrapper: Wrapper });
       expect(result.current.length).toBeGreaterThan(0);
       expect(result.current[0]).toHaveProperty('queryKey');
       expect(result.current[0]).toHaveProperty('status');
       expect(result.current[0]).toHaveProperty('dataUpdatedAt');
       expect(result.current[0]).toHaveProperty('data');
     });

     it('includes dataLength for array data', () => {
       const { Wrapper } = createWrapper();
       const { result } = renderHook(() => useCacheEntries(), { wrapper: Wrapper });
       const entry = result.current.find(e =>
         JSON.stringify(e.queryKey) === JSON.stringify(['test-key'])
       );
       // data.items is [1,2,3] so dataLength should be 3
       expect(entry?.dataLength).toBe(3);
     });

     it('returns empty array when cache is empty', () => {
       const queryClient = new QueryClient({ defaultOptions: { queries: { retry: false } } });
       const Wrapper = ({ children }: { children: React.ReactNode }) =>
         React.createElement(QueryClientProvider, { client: queryClient }, children);
       const { result } = renderHook(() => useCacheEntries(), { wrapper: Wrapper });
       expect(result.current).toEqual([]);
     });
   });
   ```
