       - `"format": "prettier --write ."`

    2. Create the Expo app:
       ```
       cd apps && npx create-expo-app mobile --template blank-typescript
       ```

    3. Install core dependencies in `apps/mobile/`:
       ```
       npx expo install nativewind react-native-reanimated react-native-safe-area-context
       npm install --save-dev tailwindcss@^3.4.17
       npx expo install expo-font @expo-google-fonts/orbitron @expo-google-fonts/inter
       npx expo install expo-linear-gradient expo-splash-screen
       npx expo install @expo/vector-icons
       npx expo install expo-router
       ```
       Note: expo-router may already be included by the template. If so, skip.

    4. Install dev tooling at root:
       ```
       npm install --save-dev eslint eslint-config-expo prettier eslint-config-prettier eslint-plugin-prettier husky lint-staged
       ```

    5. Create `apps/mobile/metro.config.js` using the NativeWind monorepo pattern from RESEARCH.md Pattern 2:
       - Use `getDefaultConfig` from `expo/metro-config`
       - Wrap with `withNativeWind(config, { input: './global.css' })`
       - Do NOT add manual `watchFolders` or `nodeModulesPaths` (SDK 52+ auto-detects)

    6. Create `apps/mobile/babel.config.js` per RESEARCH.md Pattern 6:
       - Presets: `['babel-preset-expo', { jsxImportSource: 'nativewind' }]`, `'nativewind/babel'`
       - Plugins: `['react-native-reanimated/plugin']` (MUST be last plugin)
       - Do NOT add `@worklets/babel-plugin` (Reanimated v4 includes it)

    7. Create `apps/mobile/tailwind.config.js` per RESEARCH.md Pattern 4:
       - Content: `./app/**/*.{ts,tsx}`, `./src/**/*.{ts,tsx}`
       - Preset: `require('nativewind/preset')`
       - Extended colors: primary scale (50-950, DEFAULT #2563eb), accent #06b6d4, danger #ef4444, danger-orange #f97316
       - Extended fontFamily: heading (Orbitron_600SemiBold), body (Inter_400Regular), body-medium (Inter_500Medium), body-semibold (Inter_600SemiBold)
       - Extended gradientColorStops: gradient-from #2563eb, gradient-to #06b6d4

    8. Create `apps/mobile/global.css` with Tailwind directives:
       ```css
       @tailwind base;
       @tailwind components;
       @tailwind utilities;
       ```

    9. Create `apps/mobile/nativewind-env.d.ts`:
       ```typescript
       /// <reference types="nativewind/types" />
       ```
       File MUST NOT be named `nativewind.d.ts` (conflicts with package name).

    10. Update `apps/mobile/app.json` per RESEARCH.md:
        - name: "FUELED.SHOWCASE", slug: "fueled-showcase", scheme: "fueled-showcase"
        - web: { bundler: "metro", output: "single" }
        - plugins: ["expo-router", "expo-font"]

    11. Create `apps/mobile/src/constants/colors.ts` per RESEARCH.md Colors Constants Module:
        - Export `Colors` object with primary, accent, danger, dangerOrange, gray500, background (#000000), surface, surfaceLight, border, borderLight, textPrimary (#ffffff), textSecondary
        - Export `Gradients` object with primaryButton gradient (blue to cyan)

    12. Create `.gitignore` per RESEARCH.md:
        - Exclude: node_modules/, .expo/, dist/, web-build/, .env, .env.local, .env.*.local, .claude/, .vscode/, .idea/, .DS_Store, Thumbs.db, infra/supabase/.temp/
        - Include: !.planning/

    13. Create `.prettierrc` per RESEARCH.md: semi, singleQuote, trailingComma "all", printWidth 100, tabWidth 2

    14. Create `eslint.config.js` (root, flat config) per RESEARCH.md ESLint Flat Config:
        - Uses `eslint/config` defineConfig and globalIgnores
        - Includes expoConfig and eslint-plugin-prettier/recommended

    15. Create `.lintstagedrc.json` per RESEARCH.md:
        - `apps/**/*.{ts,tsx,js,jsx}`: eslint --fix, prettier --write
        - `*.{json,md,css}`: prettier --write

    16. Set up Husky:
        ```
        npx husky init
        ```
        Update `.husky/pre-commit` to run `npx lint-staged`
