  <action>
    **Step 1: Install dependencies**
    ```bash
    cd apps/mobile && npm install react-hook-form clsx zod @hookform/resolvers
    ```

    **Step 2: Update colors.ts** — Add Gradients object with `primary` and `danger` keys as tuple arrays for LinearGradient:
    ```typescript
    export const Gradients = {
      primary: ['#2563eb', '#06b6d4'] as const,
      danger: ['#ef4444', '#f97316'] as const,
    } as const;
    ```
    Keep the existing `primaryButton` for backward compat if any code uses it, but the new pattern uses `Gradients.primary` directly with LinearGradient `colors` prop.

    **Step 3: Create Typography.tsx** — Variant-based component using clsx. Variants:
    - `h1`: `text-2xl font-heading uppercase tracking-wider text-white`
    - `h2`: `text-xl font-heading uppercase tracking-wider text-white`
    - `h3`: `text-lg font-heading uppercase tracking-wide text-white`
    - `body`: `text-base font-body text-neutral-300`
    - `caption`: `text-sm font-body text-neutral-400`
    - `label`: `text-sm font-body-semibold text-neutral-300 uppercase tracking-wider`
    - `button`: `text-base font-heading uppercase tracking-wider`
    Props: `variant`, `className`, `children`. Uses `Text` from react-native. Accepts clsx-merged className override.

    **Step 4: Create Button.tsx** — Follow logbook Button.tsx pattern exactly:
    - Variants: `primary` (gradient contained), `secondary` (bg-white/5 + border-white/15), `outline` (transparent + border-white/20), `text` (transparent, no border)
    - Sizes: `sm`, `md`
    - Props: `variant`, `size`, `disabled`, `loading`, `label`, `icon` (MaterialIcons name), `iconSize`, `onPress`, `className`
    - Primary and danger variants use `LinearGradient` with inline `style` for padding (NOT className -- NativeWind unreliable on LinearGradient). Use `Gradients.primary` / `Gradients.danger` from colors.ts.
    - Reanimated press animation: `useSharedValue(1)` -> `withTiming(0.95, {duration: 100})` on pressIn, `withTiming(1, {duration: 150})` on pressOut. Wrap Pressable in `Animated.View` with animatedStyle.
    - Use `Pressable` (NOT TouchableOpacity).
    - Disabled state: `opacity-40 cursor-not-allowed`, no onPress/onPressIn/onPressOut handlers.
    - Loading state: `ActivityIndicator` replaces icon, disabled behavior.
    - `Typography variant="button"` for label text.
    - Add `color` prop (`primary` | `danger`) for gradient selection on contained variants.

    **Step 5: Create Card.tsx** — Slot-based card:
    - Props: `header?: ReactNode`, `children: ReactNode` (body), `footer?: ReactNode`, `className?: string`, `onPress?: () => void`
    - Styling: `bg-white/5 border border-white/15 rounded-2xl overflow-hidden`
    - Header slot: `px-4 pt-4` wrapper
    - Body slot: `px-4 py-3` wrapper
    - Footer slot: `px-4 pb-4 border-t border-white/10` wrapper
    - If `onPress` provided, wrap in Pressable with Reanimated scale 0.98 press animation. Otherwise render plain View.

    **Step 6: Create Badge.tsx** — Semantic color map per user decision:
    - Types: `success`, `info`, `warning`, `error`
    - Color map: `{ success: { bg: 'bg-green-500/10', border: 'border-green-500/30', text: 'text-green-400' }, info: { bg: 'bg-blue-500/10', ... }, warning: { bg: 'bg-yellow-500/10', ... }, error: { bg: 'bg-red-500/10', ... } }`
    - Styling: `px-3 py-1 rounded-lg border` + color map classes
    - Text: `Typography variant="caption"` with uppercase, color override

    **Step 7: Create Avatar.tsx** — Circular with gradient:
    - Props: `name?: string`, `imageUri?: string`, `size?: 'sm' | 'md' | 'lg'`, `className?: string`
    - Sizes: `sm: 32`, `md: 40`, `lg: 56` (pixel values for style, not className)
    - Image mode: `Image` with `borderRadius: dim/2`
    - Initials mode: `LinearGradient` with `Gradients.primary`, circular via `borderRadius: dim/2`, white Orbitron initials centered
    - Initials extraction: first letters of name words, max 2 chars, uppercased

    **Step 8: Create Divider.tsx** — Simple horizontal rule:
    - `View` with `h-px bg-white/10` + optional `className`
    - Props: `className?: string`

    **Step 9: Create ui/index.ts barrel**:
    ```typescript
    export { default as Button } from './Button';
    export { default as Card } from './Card';
    export { default as Typography } from './Typography';
    export { default as Badge } from './Badge';
    export { default as Avatar } from './Avatar';
    export { default as Divider } from './Divider';
    ```

    **IMPORTANT:** All imports use relative paths (e.g., `../constants/colors` from components, NOT `@/`). The project does not have path aliases configured.
