  <action>
    **Table.tsx** (in ui/ per Claude's discretion — it's a data display component):
    - Props: `headers: string[]`, `rows: (string | ReactNode)[][]`, `className?: string`
    - Render as `View` (not FlatList — tables are typically small and fully rendered)
    - Header row: `View className="flex-row bg-white/5 px-4 py-3 border-b border-white/15"` with `Typography variant="label"` for each header cell. Each cell is a `View` with `flex-1`.
    - Data rows: `View className="flex-row px-4 py-3 border-b border-white/10"` with `Typography variant="body"` or ReactNode for each cell. Each cell `flex-1`.
    - Outer wrapper: `View className="border border-white/15 rounded-2xl overflow-hidden"`
    - Update `ui/index.ts` barrel to add Table export.

    **List.tsx** (in layout/):
    - Props: `items: { id: string; content: ReactNode }[]`, `onItemPress?: (id: string) => void`, `className?: string`, `divider?: boolean`
    - Use FlatList for virtualized rendering
    - Each item: `Pressable className="px-4 py-3"` (Pressable only if onItemPress provided, otherwise View)
    - If `divider` (default true): `ItemSeparatorComponent` renders `Divider`
    - Import Divider from `../ui/Divider`

    **Tabs.tsx** (in layout/):
    - Props: `tabs: { key: string; label: string }[]`, `activeTab: string`, `onTabChange: (key: string) => void`, `className?: string`
    - Underline style per user decision: horizontal row of tab labels
    - Active tab: `border-b-2 border-primary` + `text-white`
    - Inactive tab: `text-white/50`
    - Container: `View className="flex-row border-b border-white/10"`
    - Each tab: `Pressable className="px-4 py-3"` with `Typography variant="body"`
    - Typography className overridden with `text-white` or `text-white/50` based on active state

    **Accordion.tsx** (in layout/):
    - Exports `Accordion` (container) and `AccordionItem` (individual section)
    - `AccordionItem` props: `title: string`, `children: ReactNode`, `defaultOpen?: boolean`
    - Reanimated animation for expand/collapse:
      - `useSharedValue` for height multiplier (0 or 1) and chevron rotation (0 or 180)
      - `withTiming(duration: 250)` for smooth transition
      - `useAnimatedStyle` for body: `maxHeight: height.value * 500, opacity: height.value, overflow: 'hidden'`
      - `useAnimatedStyle` for chevron: `transform: [{ rotate: rotation.value + 'deg' }]`
    - Styling: `border border-white/15 rounded-2xl overflow-hidden` per item
    - Header: `Pressable` with flex-row, Typography body for title, `MaterialIcons expand-more` for chevron in `Animated.View`
    - Body: `Animated.View` wrapping content in `View className="px-4 pb-3"`
    - `Accordion` wrapper: optional, just a `View` with `gap-3` to space items

    **Skeleton.tsx** (in feedback/):
    - Exports `SkeletonLine` and `SkeletonCard`
    - `SkeletonLine` props: `width?: string | number`, `height?: number`, `className?: string`
    - `SkeletonCard` props: `lines?: number`, `className?: string`
    - Pulse animation via Reanimated: `withRepeat(withSequence(withTiming(0.3, {duration: 800}), withTiming(1, {duration: 800})), -1)` applied to opacity
    - `SkeletonLine`: `Animated.View` with `bg-white/10 rounded-lg` + animated opacity, default height 14px
    - `SkeletonCard`: `View` with card styling (`bg-white/5 border border-white/15 rounded-2xl p-4`) containing N `SkeletonLine` items with varying widths (100%, 75%, 60%) for visual variety

    **Barrel exports:**
    - `layout/index.ts`: export List, Tabs, Accordion (default + AccordionItem as named)
    - `feedback/index.ts`: export SkeletonLine, SkeletonCard (named exports, not default)

    **IMPORTANT:** Use relative imports throughout. Typography imported as `../ui/Typography`. Divider as `../ui/Divider`. No `@/` aliases.
