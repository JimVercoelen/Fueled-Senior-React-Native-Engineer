Create a new `ScrollToTop.tsx` component that provides a floating action button to scroll back to the top of a ScrollView.

**Props interface:**
```typescript
import { RefObject } from 'react';
import { ScrollView as RNScrollView } from 'react-native';

interface ScrollToTopProps {
  scrollRef: RefObject<RNScrollView | null>;
  visible: boolean;
}
```

**Implementation:**
- The component renders a circular FAB (48x48) positioned `absolute`, `bottom: 24`, `right: 24` using inline style (not NativeWind, since absolute positioning on this kind of overlay is more reliable inline).
- Background: `rgba(255,255,255,0.15)` with `borderRadius: 24`.
- Contains a `MaterialIcons` "keyboard-arrow-up" icon, size 28, color white.
- Wrapped in a `Pressable` that calls `scrollRef.current?.scrollTo({ y: 0, animated: true })` on press.
- Use Reanimated for fade in/out: `useAnimatedStyle` with opacity based on `visible` prop. Use `withTiming` for 200ms transition. Wrap the Pressable in `Animated.View`.
- When `visible` is false, also set `pointerEvents: 'none'` on the Animated.View so it doesn't block touches.

**Usage pattern** (for documentation in a comment at top of file):
```
// Usage: Add a ref and onScroll handler to your ScrollView:
//   const scrollRef = useRef<ScrollView>(null);
//   const [showTop, setShowTop] = useState(false);
//   <ScrollView ref={scrollRef} onScroll={(e) => setShowTop(e.nativeEvent.contentOffset.y > 300)} scrollEventThrottle={16}>
//   <ScrollToTop scrollRef={scrollRef} visible={showTop} />
```
